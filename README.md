# Personal Portfolio Website

This repository contains the source code for my **personal portfolio
website**, where I showcase my projects, research activities, and
professional background in Computer Science.

The website was developed to serve as a central hub for presenting my work and
academic contributions.

## How to Run Locally

1.  Clone the repository:

``` bash
git clone https://github.com/BernaTameirao/portfolio-website.git
```

2.  Access the project folder:

``` bash
cd portfolio-website
```

3.  Install the dependencies:

``` bash
npm install
```

4.  Run the development server:

``` bash
npm start
```

5.  Open in your browser:

```
http://localhost:3000
```
